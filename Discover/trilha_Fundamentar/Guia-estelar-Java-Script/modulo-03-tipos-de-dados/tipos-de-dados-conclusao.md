# Tipos de dados

Conforme o EcmaScript temos 9 tipos de dados:

* Data Types
    * Primitive / Primitive Value
    * Structural
    * Structural Primitive

## Primitivos

* String
* Number
* Boolean 
* Undefined
* Symbol
* BigInt
  
## Estuturais

  * Object
    * Array
    * Map
    * Set
    * Date
    * ...
  * Function
  
## Primitivo Estrutural / Structural Root Primitive

* Null
